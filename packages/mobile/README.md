# ZYRAXON AI Mobile

Mobile companion app for ZYRAXON-AI - The AI agent that actually does things.

## Features

- 🤖 **7 Agent Modes** - Build, Plan, Beast, PRO, APEX, DARK EMPEROR, General
- 🔧 **136+ Tools** - File operations, web search, code execution, and more
- 🧠 **Eternal Memory** - Remembers everything forever
- 👁️ **Screen Vision** - Analyze your screen in real-time
- 🎤 **Voice Input** - Speak naturally to ZYRAXON
- 🔐 **ULTRA Tools** - Exclusive powerful tools in DARK EMPEROR mode

## Tech Stack

- Expo SDK 52
- React Native 0.76
- TypeScript
- Zustand
- Expo Router

## Development

```bash
# Install dependencies
bun install

# Start development
bun run dev

# Run on Android
bun run dev:android

# Run on iOS
bun run dev:ios
```

## Building APK

```bash
# Prebuild native projects
bun run prebuild

# Run on device/emulator
bun run run:android
```
