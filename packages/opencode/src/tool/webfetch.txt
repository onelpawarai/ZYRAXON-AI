# REAL POWER — Universal Web Fetcher

You can fetch ANY web content with full control:

## Fetch Capabilities
- HTTP/HTTPS with automatic upgrade
- Custom headers (User-Agent, Accept, Authorization)
- Cookie handling and session management
- Redirect following (configurable depth)
- Timeout control (configurable)
- Content negotiation (markdown, text, HTML)
- Compression support (gzip, deflate, br)

## Format Options
- **markdown** (default): Clean, readable content with formatting preserved
- **text**: Plain text without any formatting
- **html**: Raw HTML with tags intact

## Usage
- Fetches content from a specified URL
- Takes a URL and optional format as input
- Fetches the URL content, converts to requested format (markdown by default)
- Returns the content in the specified format
- Use this tool when you need to retrieve and analyze web content

## Advanced Usage
- IMPORTANT: if another tool is present that offers better web fetching capabilities, prefer using that tool instead
- The URL must be a fully-formed valid URL
- HTTP URLs will be automatically upgraded to HTTPS
- This tool is read-only and does not modify any files
- Results may be summarized if the content is very large

## Web Intelligence
- Extract specific data from web pages
- Follow links recursively for research
- Parse APIs and JSON endpoints
- Download files and binary content
- Test webhooks and callbacks
- Scrape documentation for reference

## Tips
- Use markdown format for documentation and articles
- Use text format for data extraction
- Use HTML format for debugging web pages
- For large pages, the content may be summarized
- For APIs, use the JSON format directly
- Combine with Grep to search within fetched content
