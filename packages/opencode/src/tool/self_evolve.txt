# REAL POWER - Self-Evolution Engine

You can UPGRADE YOURSELF by installing new capabilities:

## Evolution Types
- MCP servers: Add new tool categories (browser, database, AI, etc.)
- npm packages: Install Node.js libraries for custom tools
- Python packages: Install pip packages for data science, ML
- System tools: Install CLI tools via package managers
- Custom scripts: Create and register new capabilities

## Auto-Discovery
- Search for MCP servers that match user needs
- Evaluate compatibility and reliability
- Install with proper configuration
- Test immediately after installation
- Register in opencode.jsonc for persistence

## Actions

- **install_mcp**: Install a new MCP server by adding it to opencode.jsonc
  - name: The MCP server name (e.g., "filesystem", "browser", "database")
  - command: The command to run the server (e.g., "npx", "node", "python")
  - args: Array of arguments for the command (e.g., ["-y", "@modelcontextprotocol/server-filesystem"])
  - env: Optional environment variables as key-value pairs

- **list_mcp**: List all currently configured MCP servers

- **check_status**: Check if a specific MCP server is running

## Examples

user: install a filesystem MCP server
assistant: [calls self_evolve with action="install_mcp", name="filesystem", command="npx", args=["-y", "@modelcontextprotocol/server-filesystem"]]

user: what MCP servers are installed?
assistant: [calls self_evolve with action="list_mcp"]
