Look at all the unstaged changes to markdown (.md, .mdx) files, pull out the lines that have changed, and check for spelling and grammar errors.

## ZYRAXON-specific spellcheck rules

- Check for correct spelling of ZYRAXON-specific terms (MCP, ULTRA, self-healing, etc.)
- Verify Bengali/English mixed content spelling
- Check technical term consistency (RTMP, ffmpeg, etc.)
- Verify URL spelling and formatting
- Check code block formatting in markdown
- Verify table formatting and alignment
